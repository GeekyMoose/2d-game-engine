# 2D Game / 2D Engine
> A 2D game using SDL.


# Description
This is a simple 2D game under construction.


# Dependencies
- [CMake](https://cmake.org/) (Build system)
- [SDL2](https://www.libsdl.org/)


# Build / Run (Linux Only)
> Requires C++11. Compiled and tested with g++ 6.3.0
> (Requires "pragma once" support)

```
mkdir build
cd build
camke ..
make -j4
make rungame
```


# Features
- 2D engine
    - TimeManager
    - FPS
    - Sprite Animation
    - 2D map rendering
    - Camera
- GamePlay
    - Player
- Physic
    - Simple 2D collision
